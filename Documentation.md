# SplitEasy: Simplifying Group Expense and Loan Management

## 📚 Overview  
SplitEasy is an innovative web application designed to simplify group expense sharing, personal loans, and financial tracking. By utilizing intuitive tools and an AI-powered chatbot, it empowers friends, roommates, and small groups to manage shared costs and financial interactions seamlessly.

---

## 🌟 Core Features  

### 💸 1. Expense Splitting  
- **What it Does:**  
  Simplifies splitting shared expenses like dinners, rent, or trips. It calculates and shows who owes whom with precise amounts.  
- **Key Features:**  
  - ⚙️ Automatic transaction balancing for everyone.  
  - 📊 Perfect for vacations or shared living arrangements.  
- **Example Use Case:**  
  - "Mike paid ₹1000 for dinner; Anna and Joe need to pay their shares."  
- **🎯 Benefits:**  
  Saves time, eliminates manual calculations, and avoids awkward money conversations.  

---

### 🤝 2. Loan Management Between Friends  
- **What it Does:**  
  Allows users to borrow and lend small amounts while tracking repayment schedules.  
- **Key Features:**  
  - 💼 Loan tracking with due dates and repayment statuses.  
  - 🔔 Notifications for upcoming repayments.  
- **Example Use Case:**  
  - "You borrowed ₹500 from Sam. Repayment is due in 3 weeks."  
- **🎯 Benefits:**  
  Helps maintain trust and clear records between friends.  

---

### 🤖 3. AI Chatbot Integration  
- **What it Does:**  
  A chatbot that assists users with managing finances through conversational commands.  
- **Key Functionalities:**  
  - Add expenses: "Add ₹200 for dinner with Mike."  
  - Check balances: "How much do I owe Anna?"  
  - Track loans: "Show my pending loans."  
- **🎯 Benefits:**  
  Intuitive and user-friendly for non-tech-savvy individuals.  

---

### 📈 4. Simple Credit Score  
- **What it Does:**  
  Builds a basic credit score based on repayment habits.  
- **Key Features:**  
  - Timely repayments increase the score.  
  - Delays reduce it.  
- **🎯 Benefits:**  
  Encourages financial responsibility and unlocks higher borrowing limits.  

---

### 💳 5. Seamless Payment Integration  
- **What it Does:**  
  Enables secure payments directly within the app using popular gateways.  
- **Supported Gateways:**  
  - PayPal, Stripe, and UPI.  
- **Example Use Case:**  
  - "Pay ₹300 to Joe using UPI."  
- **🎯 Benefits:**  
  Fast, secure, and eliminates the need to switch between apps.  

---

## 🛠️ Tech Stack  

### Frontend:  
- **React.js:** For a dynamic and responsive user interface.  
- **Tailwind CSS:** Ensures fast, clean, and mobile-friendly design.  
- **Chart.js:** Provides insightful visualizations for expenses.  

### Backend:  
- **Node.js with Express.js:** Manages server logic, routes, and APIs.  
- **MongoDB:** Efficiently stores user data, expenses, loans, and credit scores.  

### AI Chatbot:  
- **Dialogflow/Rasa:** Powers the chatbot for intuitive user interactions.  

### Payment Integration:  
- **Stripe/PayPal/UPI:** Enables seamless and secure transactions.  

### Authentication:  
- **JWT (JSON Web Tokens):** Secures user login and session data.
